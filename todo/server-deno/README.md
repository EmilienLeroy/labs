# todo-deno

## Prerequisite

* [Deno](https://deno.land/)
* [Mongodb](https://www.mongodb.com/)

## Install 

First, you need denon to run this project :

```
$ deno install -qAf --unstable https://deno.land/x/denon/denon.ts
```

## Run the project

To run the project use this commande :

```
$ denon run start
```
